# CSC3100-DataStructure-PersonalNotes
该仓库为针对 香港中文大学（深圳） CSC3100（数据结构） 课程整理的个人学习笔记。

当前内容主要覆盖期中考试之后的课程部分，其余章节将于后续时间逐步补充完善。

本笔记仅作为个人学习与复习的总结记录，若能配合 CSC3100 官方课件一同阅读，将有助于更系统地理解课程中的数据结构与相关算法思想。 <br><br> 

This repository contains personal study notes for CSC3100 (Data Structures) at The Chinese University of Hong Kong, Shenzhen.

At present, the notes mainly cover the course content after the midterm examination. The remaining sections will be gradually completed in the future.

These notes are intended as a personal learning and review reference. For better understanding of data structures and related algorithmic concepts, it is recommended to read them alongside the official CSC3100 lecture slides.
